## How to Run

1. Download or clone the project folder.
2. Open `index.html` in any modern browser (Chrome, Firefox, Edge, Safari).
3. No installation, no dependencies, no build step required.

> ✅ Works immediately by opening `index.html` in VS Code with Live Server or directly in a browser.

## Features Demonstrated

### HTML Concepts
- ✅ Semantic HTML5 (`<header>`, `<nav>`, `<main>`, `<section>`, `<article>`, `<footer>`)
- ✅ Forms with labels, inputs, select, textarea, checkbox
- ✅ HTML tables (skills table on About page)
- ✅ Multi-page structure with consistent navigation
- ✅ Accessibility — ARIA labels, roles, `aria-current`, `alt` text
- ✅ Meta tags for SEO and viewport

### CSS Concepts
- ✅ CSS Custom Properties (variables)
- ✅ CSS Grid (gallery, service cards, footer, process steps)
- ✅ Flexbox (navbar, hero stats, buttons, cards)
- ✅ Box Model (padding, margin, border, border-radius)
- ✅ Media Queries (mobile ≤768px, tablet ≤1024px, small ≤480px)
- ✅ Hover effects on cards, buttons, nav links
- ✅ CSS transitions and animations (progress bars, card lift)
- ✅ Glassmorphism (`backdrop-filter: blur`)
- ✅ CSS gradients (hero, buttons, progress fills)
- ✅ `@keyframes` animations (pulse dot, fillBar)
- ✅ `clamp()` for fluid typography

### JavaScript Concepts
- ✅ Mobile hamburger menu toggle
- ✅ Active navigation link detection
- ✅ Intersection Observer for scroll-triggered bar animations
- ✅ FAQ accordion (open/close with CSS max-height transition)
- ✅ Contact form submit handler with feedback
- ✅ Smooth scroll for anchor links
- ✅ Navbar shadow on scroll
