Introduction

Hello, I'm Gautam Mewara 👋

I am Gautam Mewara from Jodhpur, Rajasthan, currently pursuing B.Tech in Computer Science & Cyber Security. I am passionate about technology, problem-solving, and building modern websites that are responsive, user-friendly, and visually appealing.

My interest in web development started when I became curious about how websites are designed and developed. I began learning HTML and CSS, then gradually explored responsive design, layouts, animations, and JavaScript to create interactive web experiences.

Alongside web development, I am also learning Java and C++ through my academic coursework. These technologies have helped me strengthen my programming fundamentals, logical thinking, and problem-solving skills.

Currently, I am working as a Web Development Intern at ApexPlanet, where I am gaining practical experience by building real-world projects and improving my frontend development skills.

I enjoy learning new technologies, experimenting with modern web design concepts, and continuously improving my development abilities through projects, internships, and hands-on practice.

My goal is to become a skilled Full Stack Developer and contribute to impactful software solutions while continuously learning and growing in the technology industry.

My Path

Learning Journey & Goals

Where I started, where I am, and where I'm heading as a developer.

🌱

How It Started

My interest in web development sparked when I first inspected the source code of a website in a browser — I couldn't believe that readable text in a text editor could produce something that looked that good. I started with the basics: a simple page with a heading, a paragraph, and some colour. That was enough to get me hooked.

📚

How I'm Learning

I learn through a mix of official documentation (MDN Web Docs is my most-visited site), YouTube tutorials, and most importantly, building things. I find that reading about CSS Grid makes sense only after I've struggled to recreate a layout I admired. The struggle is where the learning actually happens.

🎯

Short-Term Goals

In the next six months, I want to become genuinely solid at JavaScript — not just DOM manipulation, but understanding how the language actually works: closures, event loops, promises, and asynchronous patterns. I'm also going to build five complete projects to put in my portfolio.

🚀

Long-Term Vision

My goal is to become a full-stack web developer within three years — someone who can build and deploy a complete web application from a database schema to a polished user interface. I see frontend as the foundation, and I plan to add Node.js, databases, and eventually a backend framework once that foundation is solid.
