Academic Java Project

Desktop Hospital Management System

A desktop-based Hospital Management System built with Java Swing for the GUI and SQLite for persistent data storage, connected through JDBC. The application manages patients, doctors, departments, rooms, and wards through a dashboard with multi-module navigation and a dark-themed interface.

Java Swing SQLite JDBC OOP DBMS
  • Patient Management
  • Doctor Management
  • Department Management
  • Room Management
  • Ward Management
  • Global Search (all tables)

Every module supports full CRUD operations (insert, update, delete, search, view), with all changes persisted to a local SQLite database. The database auto-creates and seeds sample data on first run, and the dashboard shows live stats and recent patients.

Built and run locally as a Java desktop application (Java Swing + SQLite JDBC driver) — not a hosted web app, so there isn't a live demo link. Source available on request.
