    Why Work With Me

    What Sets My Work Apart

    Small details done right. Code that's clean, sites that perform, designs that don't look like templates.

    01

    Clean, Readable Code

    Well-structured HTML and CSS with meaningful class names and proper comments — code that any developer can pick up and understand.

    02

    Genuinely Responsive

    I test on real device sizes, not just browser resize. Your site will work on the cheapest Android phone and a 4K desktop monitor.

    03

    No Unnecessary Bloat

    No Bootstrap. No React. No 3MB of dependencies for a static page. Just HTML, CSS, and the minimum JavaScript that the feature actually needs.

    04

    Accessibility First

    Semantic elements, proper ARIA labels, keyboard navigation, and colour contrast are built in from the start, not bolted on later.

    05

    Fast Learner

    Every project pushes me to learn something new. I document what I discover and apply it consistently across the entire codebase.

    06

    Open to Feedback

    I treat code review as a learning opportunity, not criticism. Revisions are part of the process, and I handle them quickly and gracefully.
