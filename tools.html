ApexPlanet Task 3

What I Built & Learned

Task 3 focused on Async JavaScript, the Fetch API, and real-world API integration.

⚡

Async / Await

Wrote modern async functions using async/await instead of callback chains, making asynchronous code cleaner and easier to reason about.

🌐

Fetch API

Used the native browser Fetch API to query the OpenWeatherMap Current Weather endpoint, parse JSON, and handle HTTP error codes explicitly.

🛡️

Error Handling

Wrapped every network call in try/catch with user-facing error messages for 404 (city not found), 401 (bad API key), and network failures.

💾

localStorage

Both apps persist state in localStorage. The Weather App remembers the last searched city; the Todo app retains all tasks including their completion status across page reloads.
